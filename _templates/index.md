# 文档模板

本目录存放各种文档模板，用于快速创建新文档。

## 模板类型

### 技术文档模板
- 基础技术文档模板
- API文档模板
- 教程文档模板

### 知识文档模板
- 研究笔记模板
- 最佳实践模板
- 问题解决模板

### 项目文档模板
- 项目计划模板
- 会议记录模板
- 总结报告模板

## 使用方法

1. 复制模板文件到目标目录
2. 根据需要修改模板内容
3. 更新YAML frontmatter元数据
4. 填充具体内容

## 维护原则

- 定期更新模板以反映最佳实践
- 保持模板的简洁性和实用性
- 为不同类型的文档提供专门模板
